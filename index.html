base

base is a starting template for some of the sites that are commonly created. This should help ramp up development time, as well as help standardize a work flow. It has been put together from several ideas and frameworks (including bootstrap and gridpak).

base has a grid system included which is based on a 12 column system with 2% padding and 1.5% gutters. It also includes several break points for mobile and small screen devices.

The sidebar is a great place to keep navs, bugs, towers, and other items.

Media breakpoints:
  • 0-479 (Small)
  • 480-959(Medium)
  • 960-1399(Large)
  • 1400+(XLarge)
